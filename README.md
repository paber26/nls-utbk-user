# Next Level Study UTBK

Frontend peserta untuk platform persiapan UTBK/SNBT berbasis Vue 3 dan Vite.

## Fitur Utama

- Dashboard persiapan UTBK dengan target skor dan roadmap belajar.
- Paket tryout UTBK yang terhubung ke API peserta.
- Halaman hasil tryout dengan ringkasan nilai dan pembahasan.
- Autentikasi peserta dan aktivasi batch/program belajar.

## Menjalankan Project

```bash
npm install
npm run dev
```

## Script

- `npm run dev` menjalankan server development Vite.
- `npm run build` membangun aplikasi untuk production.
- `npm run preview` menjalankan preview hasil build.
